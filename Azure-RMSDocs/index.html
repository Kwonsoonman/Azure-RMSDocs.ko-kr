Azure 권한 관리

Azure 권한 관리(Azure RMS)는 조직의 중요한 정보를 무단 액세스로부터 보호하고 이 정보를 사용하는 방법을 제어할 수 있도록 지원하는 클라우드 서비스입니다. Azure 권한 관리는 다른 Microsoft 클라우드 서비스 및 응용 프로그램(예: Office 365, Azure Active Directory)에 통합되었지만 온-프레미스 응용 프로그램과 서비스에도 사용할 수 있습니다.

Rights Management는 암호화, ID 및 권한 부여 정책을 사용하여 파일과 메일을 보호합니다. Rights Management를 사용하여 적용된 보호 기능은 조직, 네트워크, 파일 서버, 응용 프로그램 내부 및 외부에서 파일과 메일에 계속 적용됩니다. 데이터를 다른 사용자와 공유하는 경우에도 데이터에 대한 제어를 유지할 수 있도록 하는 정보 보호 솔루션입니다.
