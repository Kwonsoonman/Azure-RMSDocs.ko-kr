# <a name="rights-management-services-sdk-21"></a>권한 관리 서비스 SDK 2.1

## <a name="purpose"></a>용도

Rights Management Services SDK 2.1 플랫폼을 사용하면 개발자가 RMS(권한 관리 서비스) 또는 Azure Rights Management를 활용하여 정보 보호 기능을 제공하는 애플리케이션을 빌드할 수 있습니다. RMS SDK 2.1은 키 관리, 암호화 및 암호 해독 처리와 같은 복잡한 보안 실무를 처리하고 애플리케이션을 간편하게 개발할 수 있는 단순화된 API를 제공합니다.

### <a name="developer-audience"></a>대상 개발자

Microsoft 다운로드 센터의 [RMS SDK 2.1 다운로드 페이지](https://www.microsoft.com/download/details.aspx?id=38397)에서 사용 가능한 RMS SDK 2.1은 디지털 자산에 대한 권한 관리를 사용하도록 설정하고 해당 자산에 사용 약관을 적용하는 사용자 지정 애플리케이션을 만드는 데 사용됩니다. C++ 프로그래밍 언어를 알고 있어야 합니다.

질문과 대답은 [RMS FAQ](https://aka.ms/adrmsfaq)의 개발자 섹션을 참조하세요.
