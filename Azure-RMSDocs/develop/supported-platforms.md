# <a name="supported-platforms"></a>지원되는 플랫폼

이 항목에서는 권한 관리 서비스 SDK 2.1 지원 클라이언트 및 서버 플랫폼을 식별합니다.

## <a name="supported-platforms"></a>지원되는 플랫폼

-   Windows 10
-   Windows Server 2012 R2
-   Windows Server 2012
-   Windows 8
-   Windows Server 2008 R2 SP1
-   Windows 7 + SP1

**참고** 템플릿 기능 사용은 Windows Server 2008 이상에서 지원됩니다.
